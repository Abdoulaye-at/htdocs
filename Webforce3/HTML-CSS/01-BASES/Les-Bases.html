Les balises DIV :

La balise DIV, est une balise neutre de type block. Elle me servira à structurer ma page.

Elles occuppent automatiquement 100% de la largeur de ma page et agissent sur un axe vertical,elles génèrent donc un retour à la ligne.
Je pourrais toutefois changer leurs comportements avec du CSS.

Les balises Inline elles, agissentsur un axe Horizontal et donc ne génère aucun retour ligne.

Les formulaires

  • La balise form permet de créer un formulaire
  • Name:Permet de donner un nom au formulaire
  • Action: Cet attribut est OBLIGATOIRE et permet de renseigner le script,la page vers laquelle l'utilisateur sera redirigé lors de l'envoi fu formulaire. Cette page va recevoir l'ensemble des données saisie par mon utilisateur.
  • Method:Permet de précier quelle méthode [GET / POST] doit être utilisée par le navigateur pour transmettre les données saisies vers la page action.
